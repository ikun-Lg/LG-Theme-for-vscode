# 🧈 小黄油 Little Butter

一款可可爱爱的浅色 VS Code 主题：**奶油黄底 + 糖果色语法 + 腮红粉点缀**。软萌不刺眼，长时间编码舒适护眼、代码层次清晰。

## ✨ 设计理念-萌萌的type

- **奶油黄做底，可可棕落墨**：背景柔和奶油黄 (#FCF4E2)，正文用温暖可可棕 (#5A4636)，从两端降低眩光与疲劳。
- **糖果色区分语义**：每类语法一个可辨识的「深糖果」色，可爱又看得清（活跃语法色对比度 ≥4.0:1）。
- **配色取自主题图标**：奶油黄油 + 腮红粉 + 可可棕三组核心色贯穿全局，图标与主题是同一套语言。
- **交互走糖果粉**：粉色光标、粉色选区/查找、粉色焦点框；按钮/徽章用更深的草莓粉配白字。

## 🗂️ 配套文件图标主题

内置两套文件图标主题，按文件家族用糖果色编码，配奶油黄油色文件夹，覆盖 139 种扩展名 + 61 个特殊文件名：

- **小黄油 Little Butter Icons**：可爱圆角色块 + 简短标签

启用：`Cmd/Ctrl+K` → `Cmd/Ctrl+T`，在 **File Icon Theme** 里选其一。

## 🚀 安装与使用

### 方法一：VS Code 扩展市场

1. 按 `Cmd/Ctrl+Shift+X` 打开扩展面板
2. 搜索 **小黄油** 或 **Little Butter** 并安装

### 方法二：命令行安装

```bash
code --install-extension lggbond-theme
```

### 启用主题

`Cmd/Ctrl+K` → `Cmd/Ctrl+T` 打开主题选择器，选择 **Little Butter (小黄油)**。

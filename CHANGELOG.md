# v1.2.0

- **新增配套深色主题 Sunflower Dark**：暖炭底色 (#221E18，非纯黑)，与浅色共享同一套色相语言，可在浅/深之间无缝切换
- **配色全面协调化**：借鉴 Solarized / Catppuccin / Gruvbox 的感知平衡理念，告别"全是橙棕"。语法按语义分配低饱和协调色——字符串橄榄绿、关键字赤陶、函数金铜、类型青灰、数字紫梅、属性钢蓝，类别一眼可辨
- **品牌金更克制**：金黄 (#FFC300) 收归光标 / 焦点 / 徽章 / 选区等交互高亮，不再淹没在语法橙色里，品牌感反而更强
- **字重去噪**：取消关键字/函数/类的滥用粗体，仅注释与 this/参数保留斜体、Markdown 标题保留粗体，字形节奏更平稳
- **可读性达标**：所有活跃语法色经 WCAG 测算 ≥4.5:1（正文 9.5:1 / 12.2:1），注释与标点刻意置于低对比"地板"以自然弱化
- **覆盖大幅扩展**：语法作用域 13 → 78/80 组（新增参数、属性、装饰器、正则、模板插值、CSS/JSON/YAML/Markdown 细分等）；界面色 → 430/412 项（Git 装饰、minimap、peek view、括号配对着色、终端 16 色 ANSI、合并冲突等）并启用语义高亮
